The **India Climate Dashboard** is an interactive Power BI project designed to explore, analyze, and communicate daily climate patterns across India between **2013 and 2017**.  

The goal of this project was not only to analyze weather data, but to **translate raw climate metrics into an intuitive, app-like dashboard experience** that mirrors modern weather applications while remaining analytically sound.

This project emphasizes **data storytelling, UX-focused design, and analytical clarity**, making complex environmental data accessible to both technical and non-technical audiences.
